## Rough roadmap

1. Environment, FastAPI and SQLModel 
    - initial set up
    - vision and 3 year data model
    - initial Create and Read endpoints
2. Frontend Foundation (React+Vite)
    - initial web client
    - connect to backend
    - read/display vision and 3 year data
3. Complete CRUD and refinement
    - add seasons and weeks to data model
    - add relationships to data tables
    - implement Update and Delete
    - add basic security.
4. MVP web app
    - authed users
    - task tracking
    - basic scoring
5. zero knowledge encryption
    - all user data is private (opt out), except for non-personal usage telemetry (opt in)
6. QoL - user and dev
    - priority UI improvements
    - bugfixes
    - tech debt refactors
7. Initial Frontend Expansion & Tooling
    - foundation for mobile app (React Native)
    - basic DevOps with GitHub Actions and Caddy
8. Lua Neovim Plugin
9. Rust TUI
